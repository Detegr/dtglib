#dtglib
Collection of some reusable stuff that I've written. Tested on Linux/OSX/Windows.
## Concurrency
	* Threads
	* Mutexes
	* Condition variables
## Network
	* TCP Sockets + selectors
	* IpAddress -wrapper
## Packet
	* Supports different data types on one packet.
	* Network-class supports sending/receiving packets.
